# Travel Book App

This is an Android application developed with Kotlin.

## Features
- Google Maps integration to display travel locations
- Local data storage using Room Database
- Basic reactive handling with RxJava (Flowable)
- Displays travel notes and places

## Technologies Used
- Kotlin
- Android Studio
- Google Maps API
- Room Database
- RxJava
